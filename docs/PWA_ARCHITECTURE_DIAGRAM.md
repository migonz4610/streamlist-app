## Key PWA Components

### 1. Service Worker Layer
- Intercepts all network requests
- Serves cached content when possible
- Falls back to network when needed
- Caches new content automatically

### 2. Installation System
- Captures browser install prompt
- Shows custom install button
- Handles installation flow
- Detects installed state

### 3. Offline Support
- Cached app shell (HTML, CSS, JS)
- Cached assets and images
- LocalStorage for data
- Fallback pages for errors

### 4. Network Detection
- Monitors online/offline status
- Shows status to user
- Queues actions when offline
- Syncs when connection restored
